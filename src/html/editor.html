Cool features

- Add and link blocks to the selected block using Enter. Combine with Alt Shift Ctrl for other link type

- Keep Ctrl pressed when linking blocks to link multiple blocks.

- You don't need to delete a connection to connect another one

- Select multiple blocks using Ctrl or drag and drop the mouse

- You can Ctrl-Tab to switch quickly to the next tab

Differences with BehaviourTreeCreator

- You have to click a block to do anything to it. Hovering a block has no effect

- If you want to delete a connection, select the child block and press Alt-Delete

- To fold children, select the block and press M or click the link in front of the block

Have fun!
